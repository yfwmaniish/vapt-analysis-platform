# SecureSuiteX

SecureSuiteX is a premium, AI-powered Vulnerability Assessment and Penetration Testing (VAPT) platform designed to provide intelligent, modular security analysis.

## Project Structure
- `engine/`: FastAPI-based backend, asynchronous modular scanners (vortex, fuzzer, fingerprint, etc.), and LLM-driven reporting.
- `dashboard/`: Next.js frontend featuring a sleek glassmorphic UI, real-time feedback, and comprehensive report viewing.

## Security & Privacy
This repository contains the working implementation. The CLI scripts and other environment configurations are omitted.
